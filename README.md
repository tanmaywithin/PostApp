# PostApp

A simple social media-like app where users can:
- Register & login
- Create posts (text-based)
- Like/unlike posts
- View all posts
- edit the own post

## Tech Stack
- Node.js
- Express.js
- MongoDB
- EJS (templating engine)

## How to Run
1. Clone this repo:
   ```bash
   git clone https://github.com/tanmaywithin/PostApp.git
2. Install dependencies:
   npm install
3. Run the app:
   node app.js
